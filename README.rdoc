== GANATEC -> Application for management of cow farmer.

==About this Project.
Ganatec is a SW app to management cow farmer. This application cover the entired process to management cow farmer, items, medicine,buy and sell of cow, transports,etc. This app  use RFID tecnologies with a chip to identificate an animal, see an historial and suministrated medicine to the animal. This program will have a movil application make for React Native.

==API APP
This project will have API to see activities from the android and ios device, the development for react native expo project app.
